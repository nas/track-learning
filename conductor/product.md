# Initial Concept

I want you to initialize this repo as a typescript, nextjs webapp. I want to use this app as a personal learning for tracking my learning progress whether that is reading books, online courses or any other progress

## Product Overview
A personal learning tracker designed to monitor progress across books, online courses, and articles. The application provides a central dashboard to visualize current learning status and historical data.

## Core Features
- **Interactive Dashboard:** A grid-based view of all learning items with key metadata (Title, Author, Progress).
- **Visual Progress Tracking:** Elegant, gradient-filled progress bars and friendly empty states.
- **Integrated Learning Viewer:** Open associated URLs directly within the application in a dedicated split-view mode (10/90 layout).
- **CRUD Operations:** Ability to add new items, update their progress/status, and archive them when completed or paused.
- **Search & Filtering:** Real-time search by title/author and category filtering (Book, Course, Article).
- **File-based Storage:** Local persistence using JSON files, enabling repository-based data management.

## Target Experience
- **Minimalist & Fast:** Quick entry and instant updates.
- **Responsive Design:** Optimized for both desktop and mobile use.
- **Dark Mode:** Built-in support for light and dark themes.
